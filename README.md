# Angularjs testing lecture

## Testing

#### 1. Unit Tests

* single run: `npm test`
* live mode (TDD style): `npm run test-watch`

#### 1. End-to-End Tests

* start server: `npm start`
* run tests in a separate window: `npm run e2e`
